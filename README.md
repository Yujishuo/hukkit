# hukkit  
## 概况
* 主要基于java语言
* 基于papermc和自写C++的基岩版模块  
* 插件部分基于java和javascript/python语言加载，java语言插件基于bukkit(spigot,paper)和nukkit加载  
* 地图格式将使用java的地图格式  
* 基岩版部分以模块的形式进行安装，模块使用C++语言，支持多种系统和架构，不安装则只有java版部分，安装后支持java和基岩版同时游玩
